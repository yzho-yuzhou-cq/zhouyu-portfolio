Murphy 指标教学 ▸ 点击展开

MVRV = 市值 / 实现价值 = 全市场平均盈利倍数
>3 = 平均赚 200%→抛压大→顶 · <1 = 平均亏损→没人卖→底

NUPL = 1-1/MVRV = 全网未实现盈亏占比
>0.75 极度贪婪 · <0 多数人亏损（投降底部）

STH-RP = 短期持有者（<155天）平均成本
价格在线上=新人赚钱→支撑 · 跌破=新人亏损→连锁抛压
Murphy："横盘阶段最准的牛熊分界线"

Exchange Flow = 币进出交易所的净量
净流入（红）=准备卖 · 净流出（绿）=准备囤
链上数据不骗人，不管叙事说什么，看钱往哪走

ATS = 各钱包群体30天持仓变化，大户权重高
接近1=大户在积累 · 接近0=大户在派发
腾哥问"主力在干嘛"——ATS 就是量化答案

指标关系链：
MVRV/NUPL(大周期定位) → STH-RP(中周期转折) → ExFlow+ATS(微观验证)
